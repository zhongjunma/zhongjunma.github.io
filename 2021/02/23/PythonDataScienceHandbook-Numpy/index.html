Numpy 入门

Numpy 牺牲了一些灵活性，采用了固定类型的数组来高效地储存和运算。

不同于 Python lists，Numpy array 中所有元素都具有相同类型。

创建 Numpy array

  1. 将 Python list 转化为 Numpy array
  2. 利用函数构造 Numpy array，这对大型数组来说效率较高

指定 Numpy 数据类型

  1. 使用 str 指定，np.zeros(10, dtype='int16')
  2. 使用 Numpy 对象也可，np.zeros(10, dtype=np.int16)

Numpy 标准数据类型详见Numpy documentation。

Numpy array 属性

几个比较重要的属性

  1. ndim，维度数量
  2. shape，形状
  3. size，元素数量
  4. dtype，数据类型

除此之外还有

  1. itemsize，size in byte
  2. nbyte，total size in byte

基础操作

Indexing

在获取数组中的单个元素时，Numpy array 和 Python list 的操作方式几乎相同。

  1. 从0开始，负数表示从末尾开始
  2. 高维数组逗号分隔
  3. 修改时注意强制类型转换

Slicing

同样类似 Python list，遵循着 x[start: stop: step]，它们的默认值是：

  • start = 0
  • stop = size of demension
  • step = 1

注意 step 为负数的时候会将数组反转。

和 Python list 不同，Numpy array 切片返回的是 views，不是 copies。修改 views 会同时修改原数组，而修改 copies 则不会。
