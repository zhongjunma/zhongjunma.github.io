Numpy 入门

不同于 Python lists，Numpy 牺牲了一些灵活性，采用了固定类型的数组来高效地储存和运算。

Numpy array 的创建

  1. 将 Python list 转化为 Numpy array
  2. 利用函数构造 Numpy array，这对大型数组来说效率较高

几个有用的函数:

  • np.zeros()
  • np.ones()
  • np.full()
  • np.empty()
  • np.eye()
  • np.arange()
  • np.linspace()
  • np.random.random()
  • np.random.normal()
  • np.random.randint()

Numpy array 的数据类型

  1. 使用 str 指定，np.zeros(10, dtype='int16')
  2. 使用 Numpy 对象也可，np.zeros(10, dtype=np.int16)

Numpy 标准数据类型详见Numpy documentation。

Numpy array 的属性

几个比较重要的属性

  1. ndim，维度数量
  2. shape，形状
  3. size，元素数量
  4. dtype，数据类型

除此之外还有

  1. itemsize，size in byte
  2. nbyte，total size in byte

Numpy 基础操作

Indexing

在获取数组中的单个元素时，Numpy array 和 Python list 的操作方式几乎相同。

  1. 从0开始，负数表示从末尾开始
  2. 高维数组逗号分隔
  3. 修改时注意强制类型转换

Slicing

同样类似 Python list，遵循着 x[start: stop: step]，它们的默认值是：

  • start = 0
  • stop = size of demension
  • step = 1

注意 step 为负数的时候会将数组反转。

和 Python list 不同，Numpy array 切片返回的是 views，不是 copies。修改 views 会同时修改原数组，而修改 copies 则不会。

Reshape

变换数组的形状，非常常用。

Concatenation & splitting

合并数组时最需要注意的是形状的匹配以及合并的方向。合并数组的方法有：

  • concatenate，万能，注意 axis 从 0 开始，
  • vstack，纵向
  • hstack，横向
  • dstack，第三维度

分裂数组时最需要注意的断开的位置，断开 $n$ 个点会形成 $n+1$ 个数组，并且断开的点不在前面，而是包含在后面。分裂数组的方法有：

  • split，万能，注意 axis 从 0 开始，
  • vsplit，纵向
  • hsplit，横向
  • dsplit，第三维度
